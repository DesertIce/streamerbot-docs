---
title: Lens Changed
description: When the selected `Lens` changed for EpocCam
variables:
  - name: lens
    type: string
    description: The name of the `Lens` that was selected, values are either `Front` or `Back`
  - name: lensIndex
    type: number
    description: The zero based index of the `Lens` that was selected
  - name: lens
    type: string
    description: The name of the `Lens` that was previously selected, values are either `Front` or `Back`
  - name: lensIndex
    type: number
    description: The zero based index of the `Lens` that was previously selected
support:
  - name: epoccam
